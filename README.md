# Cozy Study Cat ✦

A cozy pixel-art study website with a pomodoro timer, to-do list, and a little cat companion that grows as you study.

---

## Features

- Work / Break pomodoro timer
- To-do list with add and delete
- Companion cat changes at 25%, 50%, 75%, and 100% of the session

---

HTML · CSS · JavaScript
